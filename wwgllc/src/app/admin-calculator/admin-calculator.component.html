Reward transactions are priced from HNT to $ per Coingecko API respective to the live price at the moment each reward transaction was confirmed by the blockchain, not by the current price of the coin. This means that the dollar value you recieve is not equal to the current value of the coins you mined, but rather a collection of each reward transaction's dollar value based on the coin price as each of the transactions took place.
